# AmzVest ZA — Educational Demo Project

A fictional Amazon reselling investment platform built as a UI/UX demonstration.

**⚠️ DEMO — All data, returns, and content are completely fictional.**

## Features
- Landing page with investment tiers, process steps, and testimonials
- Package selection and registration flow
- Investor dashboard with payout schedule
- Admin panel with KPIs and tabbed tables
- Dark/light mode
- Fully responsive

## Tech
- Next.js (Pages Router) + TypeScript + Tailwind CSS
- framer-motion, lucide-react, next-themes

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
