# AmzVest ZA — Educational Demo Project

A fictional Amazon reselling investment platform. Built as a UI/UX demonstration.

## Key facts
- All data, figures, testimonials, and returns are completely fictional
- Every page displays a DEMO disclaimer
- No real authentication, no real payments, no real investments
- Static demo — no backend, no database, no API calls

## Tech
- Next.js (Pages Router) + TypeScript + Tailwind CSS
- framer-motion for animations
- lucide-react for icons
- next-themes for dark/light mode

## Pages
- `/` — Landing (hero, trust bar, how it works, packages, testimonials, FAQ, disclaimer)
- `/packages` — Investment package tiers
- `/how-it-works` — Process explanation
- `/faq` — Frequently asked questions
- `/contact` — Contact form (demo)
- `/login` — Sign in (demo)
- `/register` — Create account with package selection (demo)
- `/dashboard` — Investor dashboard (demo)
- `/admin` — Admin login (demo)
- `/admin/dashboard` — Admin panel with KPIs and tabbed tables (demo)
